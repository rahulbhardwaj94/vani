# Vani demo video — 45-second shooting script

**Tools:** QuickTime (File → New Screen Recording, capture mic audio) or
[Screen Studio](https://screen.studio) (auto-zoom makes it look pro).
Record at 1080p+, trim to ≤45 s. Export vertical crop for Reddit/Twitter,
horizontal for LinkedIn/YouTube.

**Setup before recording:** Wi-Fi OFF (menu bar visible — this is the money
shot for "fully local"). Open Notes with a large font. Vani running.

| Time | On screen | You say (voiceover or live dictation) |
|---|---|---|
| 0–4 s | Menu bar close-up: Wi-Fi icon OFF, Vani mic icon visible | "This is my Mac. No internet." |
| 4–14 s | Notes app. Hold Right ⌥ — pill appears, bars dancing. Dictate live: | "This entire sentence is being transcribed on my laptop by open-source AI, with no cloud, no account, and no subscription." |
| 14–18 s | Text lands in Notes. Beat. Cursor blinks. | (silence — let it land) |
| 18–26 s | Hold ⌥ again, dictate in Hindi: | "मेरा नाम राहुल है और यह पूरी तरह से मेरे लैपटॉप पर चल रहा है।" — text appears in Devanagari |
| 26–34 s | Open menu bar → History & Vocabulary. Scroll history briefly. | "Everything I've ever dictated stays here. On disk. Mine." |
| 34–42 s | Quick cut: GitHub repo page (README hero) | "It's called Vani — Sanskrit for 'voice.' Fifteen hundred lines of Swift, MIT licensed, link below." |
| 42–45 s | End card: repo URL + वाणी | (music out) |

**Three rules:** dictate LIVE on camera (screen recordings of real latency
build trust — don't speed it up); keep the Wi-Fi-off icon visible in every
menu-bar shot; end on the URL for 3 full seconds.
